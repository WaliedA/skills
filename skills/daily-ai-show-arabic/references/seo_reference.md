# YouTube SEO Reference — يومياتي مع الذكاء الاصطناعي

## Title Optimization

YouTube titles in Arabic should be 50-60 characters. The algorithm favors titles that:
- Start with the most searchable keyword
- Include a number when possible
- Create a curiosity gap
- Mix Arabic and English keywords for bilingual discovery

### Title Templates (pick the best fit each day)

**Template A — Lead Story Focus:**
```
[أقوى خبر بـ 5-7 كلمات] | يومياتي مع الذكاء الاصطناعي
```
Example: نموذج Nvidia الجديد يتفوق على الجميع | يومياتي مع الذكاء الاصطناعي

**Template B — Number + Curiosity:**
```
[عدد] أخبار ذكاء اصطناعي [صفة] اليوم | [التاريخ]
```
Example: 8 أخبار ذكاء اصطناعي صادمة اليوم | 30 مارس 2026

**Template C — Question Hook:**
```
[سؤال مثير]؟ + أهم أخبار AI اليوم
```
Example: هل تنتهي شراكة OpenAI ومايكروسوفت؟ + أهم أخبار AI اليوم

## Tag Strategy

### Evergreen Tags (use every episode):
```
ذكاء اصطناعي, أخبار الذكاء الاصطناعي, AI news, artificial intelligence, أخبار التقنية, تقنية, تكنولوجيا, يومياتي مع الذكاء الاصطناعي, AI Arabic, tech news Arabic, أخبار AI, ذكاء اصطناعي بالعربي, AI news today, daily AI news
```

### Rotating Tags (change per episode based on content):
- Company names: OpenAI, Google AI, Anthropic, Nvidia, Meta AI, xAI
- Model names: GPT, Claude, Gemini, Llama, etc.
- Topic tags: AI agents, LLM, وكلاء ذكاء اصطناعي, نماذج لغوية, open source AI
- Regional: تقنية الإمارات, AI UAE, السعودية تقنية, Gulf tech

## Thumbnail Text Rules

1. **Max 4-5 Arabic words** — must be readable on a phone screen
2. **Use thick, bold Arabic font** (suggest: Cairo Bold or Tajawal Black)
3. **High contrast colors**: white text on dark, or yellow/red text for urgency
4. **Include one English keyword** if relevant (company name or tech term)
5. **Emotion triggers**: 🔥 صدمة, ⚠️ تحذير, 🚀 اختراق, 💰 صفقة ضخمة

### Thumbnail Template Options:

**Option A — Shock/Breaking:**
```
Text: "[شركة] غيّرت كل شي!"
Visual: Company logo large, red/orange gradient background, shocked emoji or presenter face
```

**Option B — List/Number:**
```
Text: "[عدد] أخبار AI مهمة"
Visual: Grid of small logos/icons, blue-purple tech gradient
```

**Option C — Question/Curiosity:**
```
Text: "نهاية [شركة/تقنية]؟"
Visual: Dramatic image of the subject, question mark overlay, dark mood
```
